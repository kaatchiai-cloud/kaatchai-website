FAQ

Common questions

Do I need a Gemini API key to use Kaatchi AI? +
On the Free plan, yes — you'll need your own Google Gemini API key (free to get). On the Pro plan, we provide managed API access so you don't need to worry about keys or limits.
What languages are supported for voiceover? +
We support 20+ languages including all major Indian languages (Hindi, Tamil, Telugu, Kannada, Malayalam, Bengali, Marathi, Gujarati, Punjabi) plus English, Spanish, French, Arabic, Portuguese, Indonesian, Japanese, Korean, German and more.
How long does it take to generate a reel? +
Typically 60–90 seconds for a complete 60-second reel — including script, voiceover, AI images and export. Batch mode lets you generate dozens simultaneously.
Can I add subtitles in a different language? +
Yes! Kaatchi AI can auto-translate subtitles to any supported language. So you can narrate in Tamil and show English subtitles, or narrate in Hindi with Tamil subtitles — great for multilingual audiences.
What format is the exported video? +
Videos export as MP4 (1080×1920, portrait 9:16) — ready to upload directly to YouTube Shorts, Instagram Reels, or TikTok. WebM fallback available on some browsers.
Is there a limit on how many reels I can make? +
On the Free plan with your own API key, there are no limits on our end — your API usage costs apply. On the Pro plan, we provide generous monthly limits with the managed API included.
